# Taglines

Short one-liners grouped by character budget, for directory listings, app store
fields, social bios, and the like. Pick the longest one that fits the field's
limit. Counts in brackets are exact character counts including spaces.

Every line is built from [`facts.md`](facts.md). Don't add a claim that isn't
there. No em dashes.
