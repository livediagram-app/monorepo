The AI panel has two modes, picked from tabs at the top: **Ask** and **Clean**. Both take the same prompt box, but they treat the diagram differently. Ask only reads the diagram; Clean changes it.

## Ask — questions, no changes

**Ask** answers questions about the diagram without touching it. Use it to get your bearings on a diagram someone else made, or to check your own: "what happens if the payment fails?" or "which steps have no outgoing arrow?". The answer streams into the panel as text, and nothing on the canvas changes.

## Clean — tidy what's there

**Clean** works on what is already on the tab. It fixes label typos and normalises sizes, positions, and styles so a diagram that grew messily reads consistently. Clean respects your layout rather than re-flowing it, so it is the safe choice once a diagram is arranged the way you like. It applies as a single block, so one **Undo** (Ctrl+Z) reverses the whole operation.

## Focus and context

Whatever the mode, the assistant only ever sees the **active tab**. If you have elements selected when you send a request, those are marked as the focus: Clean concentrates its edits there while still seeing the rest of the tab, so an arrow whose other end sits outside the selection stays connected correctly, and Ask answers about the selection in context. With nothing selected, the whole tab is fair game.

<Note>
  Prompts are kept short by design, so think of each request as one clear instruction. If a result
  is not quite right, adjust the prompt and send again rather than stacking corrections into one
  long message.
</Note>
