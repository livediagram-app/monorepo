Sometimes you want a shape to stand out, or a set of arrows to look consistent, without fiddling with fill, border, weight, and pattern one control at a time. Style presets give you a **Presets** category in the selected-element context menu with ready-made looks you can apply in a single click. The category sits at the top of the appearance group, just above Animation, and presets always track your active theme so they stay on-brand.

## Shape presets

Right-click a single shape and open the **Presets** category to find two rows you combine freely, plus a reset:

- **Colour** offers eight one-click colour variations derived from the active theme. Each sets the shape's fill, border, and text colour together, spanning the on-theme look plus soft, tinted, solid, bold, outline, muted, and inked emphasis variants. On multi-colour themes you also get a card per branch hue. Text colour is auto-contrasted on filled variants so labels stay readable.
- **Border** offers up to eight border variations that combine weight (thin / medium / thick), pattern (solid / dotted / dashed), and corner radius (none / small / medium / large / full), including a sharp un-rounded preset and a rounded one.

Colour and border presets are independent: applying one never disturbs the other, so they combine. You might pair a tinted colour with a dashed, un-rounded border, for example. **Reset to default** clears the shape's colour and border overrides, returning it to its theme default in one click.

## Arrow presets

Right-click a single arrow and the **Presets** category offers eight one-click arrow styles. Each combines a line pattern (solid / dashed / dotted), a thickness, and an optional flow animation, so you get looks like a dashed animated arrow or a travelling-dot arrow. **Reset to default** clears the arrow's pattern, thickness, and animation overrides.

## Good to know

- The Presets category appears only when a single element is selected, and only the matching set shows: shape presets for a shape, arrow presets for an arrow.
- Each preset applies as one undo step, so you can revert a look in a single press.
- Because colour presets are theme-derived, switching the diagram's theme refreshes the available colours to match.

<Tip>Use a bold or inked colour preset to draw the eye to the one shape that matters, then leave the rest on the theme default for contrast.</Tip>
