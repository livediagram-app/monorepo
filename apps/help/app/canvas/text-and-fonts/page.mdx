Text is what turns a box into a step and an arrow into a relationship. Almost every element in livediagram can carry a label, and you can shape how that text reads by choosing its typeface from a curated set of eight fonts, ranging from crisp sans-serifs to a relaxed handwriting face. Fonts can be set on a single element or as a default for an entire tab.

## Editing labels

To add or change a label, edit the element directly on the canvas and type. Shapes, text elements, sticky notes, table cells, and arrows can all carry text, so you can annotate a diagram wherever you need to. Edits go through the normal history path, so undo and redo cover label changes like any other edit.

## Choosing a typeface

livediagram ships eight Google Fonts, picked to give a diagram a distinct voice: Inter, Roboto, Poppins, Nunito, Lora, Roboto Slab, Roboto Mono, and Caveat. Together they span crisp, friendly, formal, and hand-drawn looks. See [Choosing fonts](/help/canvas/text-and-fonts/choosing-fonts/) for where to set them and how the levels interact.

## Per element or per tab

Fonts work at two levels. You can set a font on a single element so one label stands out, or set a default for the whole tab so every label shares the same typeface unless it overrides it. The tab default also covers a default text size for new elements, keeping a fresh canvas tidy. [Choosing fonts](/help/canvas/text-and-fonts/choosing-fonts/) explains how an element falls back to the tab's font when it has none of its own.

<Tip>
  Switching the [theme](/help/canvas/themes/) only recolours your diagram, it never changes your
  fonts, so the typefaces you pick stay put when you restyle.
</Tip>
