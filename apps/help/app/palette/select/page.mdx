Select is the default mode for working on the canvas, and the one you spend most of your time in. You can switch to it from the mode picker at the top of the floating palette, or press **`V`**.

## What you can do

In Select mode the pointer interacts directly with your elements:

- **Click** an element to select it.
- **Drag a marquee** across empty canvas to select many elements at once.
- **Drag** an element to move it.
- **Drag its handles** to resize.
- **Double-click** to edit a label.

## Returning here

Select is your home base. After you finish with any other tool (Hand, Eraser, the Format Painter, and so on), come back to Select to carry on editing. Choosing a different tool changes what a click or drag does, so returning to Select restores normal selection and editing behaviour.

<Tip>Press `V` from anywhere to jump straight back to Select without opening the mode picker.</Tip>
