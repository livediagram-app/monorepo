The Format Painter mode copies the look of one element (its colours, line, and text styling) and applies it to others, so a set of shapes can share a consistent style without setting each one by hand. Choose it from the mode picker at the top of the floating palette.

## How to use it

<Step n={1}>Pick the Format Painter mode from the palette.</Step>
<Step n={2}>Click a source element to copy its look.</Step>
<Step n={3}>Click each element you want to apply that style to.</Step>

Every element you click takes on the copied colours, line, and text styling.
