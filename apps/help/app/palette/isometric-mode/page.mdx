Isometric Mode toggles the current tab into a tilted, isometric (3D-looking) perspective, giving a flat diagram a sense of depth. Choose it from the mode picker at the top of the floating palette, or press **`I`**.

## How it works

The mode restyles how the whole tab is rendered, tilting it into the isometric view. Toggle it again to return to the flat view. It is a toggle for the current tab, so it affects everything on that tab at once.

## When it is unavailable

Isometric Mode is disabled on an empty canvas. Add an element and the mode becomes available again.
